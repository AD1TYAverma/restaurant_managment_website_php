# Restaurant Management System

A full-stack restaurant management web application built using HTML, CSS, JavaScript, PHP, and MySQL. It helps streamline restaurant operations like order handling, billing, and user management.

## Features
- User Authentication (Login/Signup)
- Menu Management System
- Order Processing
- Admin Dashboard
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Modules
- User Module (Login/Register)
- Admin Module (Manage Menu & Orders)

## Purpose
This project is developed for learning full-stack web development and real-world application building.
